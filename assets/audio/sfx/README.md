# SFX Assets

Sound effects for game interactions.

## Required Files

### Dice
- `dice-click.mp3` - Click/tap on dice
- `dice-roll.mp3` - Dice rolling sound
- `dice-land.mp3` - Dice landing/settling

### Luck/Score
- `luck-sparkle.mp3` - Small luck gain
- `luck-jackpot.mp3` - Big luck gain (jackpot)

### Purchases
- `purchase-success.mp3` - Successful purchase
- `purchase-fail.mp3` - Failed purchase (not enough currency)

### Achievements
- `achievement-unlock.mp3` - Achievement unlocked
- `level-up.mp3` - Level up fanfare

### UI
- `ui-open.mp3` - Panel/modal opening
- `ui-close.mp3` - Panel/modal closing

## Format

- MP3 format recommended
- Short duration (< 2 seconds typically)
- Normalized volume levels
