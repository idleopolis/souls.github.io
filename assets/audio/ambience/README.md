# Ambience Assets

Ambient sound loops for environmental atmosphere.

## Required Files

- `wind-light-loop.mp3` - Gentle wind sound (looping)
- `wind-strong-loop.mp3` - Strong/howling wind sound (looping)
- `sleigh-bells.mp3` - Distant sleigh bells (one-shot, periodic)

## Format

- MP3 format recommended
- Should loop seamlessly (except one-shot sounds)
- Keep volume consistent across files
